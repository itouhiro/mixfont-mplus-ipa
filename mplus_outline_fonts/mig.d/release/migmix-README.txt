概要
----

MigMix（ミグミックス）フォントは、以下の2種類を
合成したTrueTypeフォントです。

* M+（エムプラス）フォント
* IPA（アイピーエー）ゴシック

M+のすべての文字を含み、
不足する漢字をIPAゴシックで補いました。



特徴
----

・太字（ボールド体）を用意しました。
　太字に、自動生成（擬似ボールド）ではなく、用意された字形が使われます。
